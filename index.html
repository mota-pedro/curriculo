Projetos

  • github.com/mota-pedro/TechCycle

    O repositório TechCycle contém o front-end de uma aplicação em Flutter para reciclagem de lixo eletrônico que utiliza a API do Google Maps. O projeto facilita a localização de pontos de coleta, promovendo a sustentabilidade através do descarte correto de dispositivos eletrônicos, com uma interface intuitiva para o usuário.

  • github.com/mota-pedro/poomodoro

    Este repositório contém uma aplicação simples de timer Pomodoro desenvolvida em JavaScript, HTML e CSS. A ferramenta ajuda a gerenciar o tempo de estudo ou trabalho com ciclos de 25 minutos de foco seguidos por breves intervalos.

  • github.com/mota-pedro/ArtAudio

    ArtAudio é uma extensão Chrome que transforma texto em áudio, desenvolvida com JavaScript, HTML e CSS. Facilita a acessibilidade ao permitir que usuários escutem conteúdos de páginas da web, com controles simples de reprodução e ajuste de velocidade.
